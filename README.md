# Business Overview – Dina Bakery

# 🥐 Dina Bakery

## 1. Nama Bisnis

**Dina Bakery**

---

# 2. Deskripsi Bisnis

Dina Bakery merupakan usaha yang bergerak di bidang penjualan roti, croissant, pastry, dan minuman secara online. Website e-commerce ini dibuat untuk memudahkan pelanggan melihat katalog produk, melakukan pemesanan, memilih metode pembayaran, serta memperoleh informasi mengenai promo yang sedang berlangsung.

Dina Bakery berkomitmen menghadirkan produk berkualitas tinggi dengan bahan-bahan segar, rasa premium, harga terjangkau, serta pelayanan yang cepat dan ramah.

---

# 3. Value Proposition

Dina Bakery memberikan berbagai nilai tambah bagi pelanggan, yaitu:

* Produk dibuat menggunakan bahan berkualitas.
* Roti selalu diproduksi dalam kondisi fresh setiap hari.
* Harga terjangkau untuk semua kalangan.
* Pemesanan dapat dilakukan secara online.
* Tampilan website sederhana dan mudah digunakan.
* Banyak pilihan metode pembayaran digital.
* Promo dan diskon tersedia secara berkala.

---

# 14. Simulasi Payment Gateway (Midtrans)

Website menggunakan simulasi **Midtrans Sandbox**.

Alur pembayaran:

* Checkout
* Sistem mengirim data transaksi
* Midtrans menampilkan halaman pembayaran
* Pelanggan memilih metode pembayaran (QRIS, Virtual Account, E-Wallet)
* Pembayaran berhasil
* Status transaksi berubah menjadi "Paid"
* Pesanan diproses oleh admin

Status pembayaran:

* Pending
* Success
* Failed
* Expired

---

# 15. Rencana SEO

Optimasi SEO dilakukan dengan:

* Menggunakan kata kunci seperti "Bakery Bandung", "Croissant Premium", dan "Roti Fresh".
* Menambahkan meta title dan meta description.
* Menggunakan heading (H1–H3).
* Optimasi gambar dengan atribut **alt**.
* URL yang sederhana dan mudah dibaca.
* Website responsif.
* Kecepatan loading yang baik.

---

# 16. Keamanan Website

Keamanan yang diterapkan meliputi:

* HTTPS/SSL.
* Validasi input pengguna.
* Sanitasi data.
* Perlindungan dari SQL Injection.
* Perlindungan terhadap Cross Site Scripting (XSS).
* Backup database secara berkala.
* Password admin dienkripsi.

---

# 17. Pemeliharaan Website

Pemeliharaan dilakukan secara rutin dengan:

* Memperbarui katalog produk.
* Menghapus produk yang sudah habis.
* Memperbarui harga.
* Memperbaiki bug.
* Backup data.
* Monitoring performa website.
* Memperbarui sistem keamanan.

---

# 18. Rencana Penggunaan Data Analytics

Data analytics digunakan untuk membantu pengambilan keputusan bisnis.

Data yang dianalisis meliputi:

* Produk paling laris.
* Produk yang jarang dibeli.
* Jumlah transaksi harian.
* Pendapatan bulanan.
* Nilai rata-rata pembelian pelanggan.
* Tingkat konversi pengunjung menjadi pembeli.
* Performa promo dan diskon.
* Metode pembayaran yang paling sering digunakan.

Hasil analisis tersebut dimanfaatkan untuk:

* Menentukan stok produk.
* Menentukan strategi promosi.
* Menyesuaikan harga.
* Menambah produk baru.
* Meningkatkan pengalaman pelanggan.
* Meningkatkan penjualan dan keuntungan perusahaan.

---

# Kesimpulan

Dina Bakery merupakan bisnis bakery berbasis e-commerce yang menawarkan kemudahan berbelanja secara online dengan produk berkualitas, harga kompetitif, serta dukungan strategi pemasaran digital, pembayaran online, keamanan website, SEO, dan analisis data. Dengan penerapan strategi tersebut, Dina Bakery diharapkan mampu meningkatkan kepuasan pelanggan, memperluas pasar, dan meningkatkan pendapatan secara berkelanjutan.
